# Alfred

## alfred常用快捷键

| 功能 | 快捷键 |
| --- | :---: |
| 用默认方式打开文件 | ⌘O |
| 用默认的搜索引擎搜索 | ⌃↩︎ |
| 在访达中显示 | ⌘↩︎ |
| 打开文件1 | open |
| 打开文件2 | 空格键 |
| 在访达中查找文件 | find |
| 查找文件内容 | in |
| 打开废纸篓 | trash |
| 清空废纸篓 | emptytrash |
| 查单词 | define |
| 休眠 | sleep |
| 关闭显示器 | sleepdisplays |
| 关机 | shutdown |
| 登出 | logout |
| 重启 | restart |
| 锁屏 | lock |
